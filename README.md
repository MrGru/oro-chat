oro-chat
========

Simple peer to peer video chat using [meteor.js](https://www.meteor.com/) and [peer.js](http://peerjs.com/).

Local example
-------------

Install Meteor:

```sh
$ curl https://install.meteor.com | /bin/sh
```

Run oro-chat:

```sh
$ meteor
```

The application will be available on `http://localhost:3000`
